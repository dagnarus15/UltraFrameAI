# Security Policy

## Supported Versions

Security fixes are handled for the latest public release and the current `main` branch.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| Older versions | No |

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Use one of these options:

- open a private GitHub Security Advisory for this repository, if available;
- contact the author through https://dagnaruscode.eu.

When reporting, please include:

- affected version or commit;
- steps to reproduce;
- expected and actual behavior;
- potential impact;
- any relevant logs or screenshots.

We will try to confirm valid reports and prepare a fix as soon as practical.
